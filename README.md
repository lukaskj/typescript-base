# TypeScript base project
## Boilerplate project using:
- TypeScript
- Routing Controllers
- TypeORM
- Axios
- ESLint


## Commands:
- `yarn dev` -> Start development server
- `yarn build` -> Build project
- `yarn start` -> Start project previously built
- `yarn lint` -> Linter
- `yarn typeorm` -> Linter with automatic fixes
